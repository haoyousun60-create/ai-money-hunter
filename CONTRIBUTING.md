# Contributing to AI Money Hunter

Welcome! This repo is built for developers who want to earn money through code bounties, airdrops, and AI-powered side hustles.

## How to Contribute

### Add New Platforms
Found a new bounty platform or airdrop opportunity? Submit a PR!

1. Fork the repo
2. Add the platform to the appropriate table in README.md
3. Include: name, URL, potential earnings, difficulty, and any gotchas
4. Open a PR with title `[NEW PLATFORM] <Platform Name>`

### Add Case Studies
Completed a bounty or airdrop? Share your experience!

1. Add your case study to the "Real-World Case Studies" section
2. Include: platform, payout (or "pending"), timeline, key learnings
3. PR title: `[CASE STUDY] <What you did>`

### Fix Errors
Found outdated info? PRs welcome for corrections.

## Contribution Standards

- **Be specific**: "Earnings $500-$2,000" is better than "earn money"
- **Include URLs**: Every platform needs a direct link
- **Verify before adding**: Don't add unverified platforms
- **No spam**: This is a curated list, not a link dump

## Format for New Platforms

```
| [Name](URL) | Type | Potential | Difficulty | Notes |
|-------------|------|-----------|------------|-------|
| Example | AI tool | $100-$1,000/mo | ⭐⭐ | Notes here |
```

## Format for Case Studies

```
### Case Study: <Title>
**Platform:** <Name>
**Payout:** $<amount> (or "pending" or "rejected")
**Timeline:** <X days/weeks>
**What I did:** <Brief description>
**Key insight:** <What you learned>
```
